# Ford Car Price Prediction

This project predicts used Ford car prices using Machine Learning.

## Features
- Year
- Mileage
- Tax
- MPG
- Engine Size
- Transmission
- Fuel Type
- Car Model

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit

## Model Performance
- R² Score: 0.83

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
